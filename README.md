# Employee_Attrition_Project

# 🏢 Machine Learning-Based Employee Attrition Prediction & Risk Scoring System

## 📌 Project Overview

Employee attrition is a major challenge for organizations as it affects productivity, recruitment costs, and workforce planning. This project develops a **Machine Learning-based Employee Attrition Prediction and Risk Scoring System** that predicts whether an employee is likely to leave the organization and provides a risk classification.

The system combines machine learning, data analytics, and an interactive Streamlit dashboard to help HR teams identify high-risk employees and make data-driven retention decisions.

---

## 🎯 Objectives

* Predict employee attrition using machine learning algorithms.
* Analyze factors influencing employee turnover.
* Calculate employee risk levels.
* Provide an interactive HR analytics dashboard.
* Support proactive employee retention strategies.

## ⚙️ Installation & Setup

### 1. Clone Repository

```bash
git clone <[your-github-link](https://github.com/priyadharshinivetrivel/Employee_Attrition_Project.git)>
```

### 2. Install Required Libraries

```bash
pip install -r requirements.txt
```

### 3. Run Streamlit Application

```bash
streamlit run app.py
```

The dashboard will open in your browser.
